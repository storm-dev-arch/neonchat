Политика конфиденциальности

Мы уважаем вашу приватность. Данный прототип хранит данные (пользователи, сообщения) в памяти сервера и удаляет их при перезапуске.

Какие данные собираются

  • Номер телефона для входа
  • Сообщения, отправленные в общий чат
  • Техническая информация соединения WebSocket

Как используются данные

  • Для аутентификации и отправки сообщений
  • Для обеспечения работы чата в реальном времени

Хранение

Данные в демо не сохраняются на диск. Для продакшена используйте БД и зашифрованное хранилище секретов.
